ATM project in C programming language that showcases the fundamental functionalities of an ATM system. 
The project enables users to perform essential operations such as depositing and withdrawing money from their accounts. 
The program incorporates a user-friendly interface, allowing customers to interact with the ATM using simple prompts and commands.

To add money to their account, users can input the desired amount and select the deposit option. 
The program validates the transaction, updates the account balance, and provides a confirmation message. 
Similarly, users can withdraw funds by entering the withdrawal amount and selecting the appropriate option. 
The program verifies the availability of sufficient funds and completes the transaction if the conditions are met.
